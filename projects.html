Mentor Match

Description:
The MentorMatch project was an ASP.NET C# MVC 5 Entity Framework 6 project that was built to help support the operations of the OSU Medical Center's Pilot Mentor Match Program. The basis for the program was to match experenced doctors (Mentors) with one or mutiple inexperienced doctors in order to allow a transfer of knowledge through gained expeience.
